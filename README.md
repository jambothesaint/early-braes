# Early Braes

Community Home Support Service – Glasgow

Early Braes is a small, community-focused home support service providing companionship, sitting visits, home cleaning, and combined support for older people living at home.

This repository contains the single-page Bootstrap website used to present the service to families, carers, and collaborators.

---

## 🌿 About Early Braes

Early Braes was created to offer calm, reliable and respectful support for older people who want to remain comfortable in their own homes.

We currently provide:

* **Companionship & Sitting Visits**
* **Home Cleaning Support**
* **Combined Support Visits (Companionship + Cleaning)**

We do **not** currently provide:

* Personal care (washing/dressing)
* Medication administration

---

## 💻 Website Overview

This is a responsive Bootstrap 5 single-page site.

Features include:

* Mobile-friendly layout
* Hero section with background image
* Three clear service cards
* Trust and boundary messaging
* Work with us section
* Contact section
* Simple placeholder enquiry form

## 🚀 Deployment (GitHub Pages)

1. Create a new repository on GitHub.
2. Upload `index.html` and the `images` folder.
3. Go to **Settings → Pages**.
4. Under "Source", select:

   * Deploy from branch
   * Branch: `main`
5. Save.

Your site will be live within a few minutes.

---

## 🔧 Customisation

To edit the site:

* Update text directly inside `index.html`.
* Replace images inside the `/images` folder.
* Update phone number and email in the Contact section.

The site uses Bootstrap via CDN, so no additional setup is required.
